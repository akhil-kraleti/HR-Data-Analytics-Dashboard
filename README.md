# ЁЯУК HR Data Analytics Dashboard (Power BI)

## ЁЯЪА Project Overview

The **HR Data Analytics Dashboard** is an interactive Power BI project designed to provide comprehensive insights into workforce demographics, salary distribution, hiring trends, employee performance, and departmental analytics. The dashboard enables HR teams and business leaders to make data-driven decisions by visualizing key workforce metrics across multiple dimensions.

This project demonstrates skills in **data cleaning, transformation, data modeling, DAX calculations, and interactive dashboard development using Power BI**.

---

## ЁЯОп Objectives

- Analyze overall workforce distribution across departments.
- Monitor employee salary trends and departmental spending.
- Evaluate employee performance ratings.
- Understand hiring patterns over time.
- Compare workforce metrics between countries.
- Provide interactive filtering for deeper HR insights.
